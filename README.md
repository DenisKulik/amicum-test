## Built With
This is a AMICUM test project built using  `Javascript`, `Vue`, `Vue Router`, `Vuex`, `HTML` and `SCSS`.

## Project setup
```
npm install -g pnpm
```

```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm run serve
```

### Compiles and minifies for production
```
pnpm run build
```

### Lints and fixes files
```
pnpm run lint
```
